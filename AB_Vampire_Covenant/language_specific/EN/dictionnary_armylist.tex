% Shrieking Horror

\newcommand{\chillingshriek}{Chilling Shriek}
\newcommand{\chillingshriekdef}{%
	A model with this Special Attack can use it as a Shooting Attack and as a Special Attack when Engaged in Combat.%
	\begin{itemize}%
		\item As a Shooting Attack with March and Shoot. Choose a target using the normal rules for Shooting Attacks. The attack has \range{8}.%
		\item As a Special Attack when Engaged in Combat. The attack is made at the model part's Agility. Declare that you are using Chilling Shriek when allocating attacks. If used, the model part cannot make Close Combat Attacks. Choose a single unit in base contact as the target.%
	\end{itemize}%
	Regardless of whether it is used as a Shooting or Melee Attack, the Chilling Shriek causes 1 hit to the target for each Health Point the model with Chilling Shriek currently has. These hits \textbf{always} have Strength 10, Armour Penetration 10, and \textbf{Magical Attacks}. When rolling to wound with this attack, use the enemy's Discipline instead of its Resilience.%
}
